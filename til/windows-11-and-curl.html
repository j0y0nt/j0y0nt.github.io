Windows 11 curl = Powershell Invoke-WebRequest.

To make post request

Set headers first on command prompt.

$headers = @{ >> "Content-Type" = "application/json" >> "Authorization" = "Bearer YOUR_ACCESS_TOKEN" >> }

Then send the POST request by running below command:

Invoke-WebRequest -Uri "http://127.0.0.1:53251/transactions/async/444666" -Method POST -Headers $headers -Body "5.63"
