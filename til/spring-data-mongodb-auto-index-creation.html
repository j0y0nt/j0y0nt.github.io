MongoDB index auto creation and spring boot test containers.

Today a simple unit test case to throw DuplicateKeyException failed, when item with same value is inserted. Apparently in Testcontainers you have to enable following property so that MongoDB create unique index for entity(ies): spring.data.mongodb.auto-index-creation=true
