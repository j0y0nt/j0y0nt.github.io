Trailing slash in Django URL

I am also learning python/django for a while now, and using book "Test-Driven Development with Python". Below is simple method to return a list of items:
def view_list(request, list_id): list_ = List.objects.get(id=list_id) return render(request, 'list.html', {'list': list_})
And below is the test cases which, puzzled me by failing for half an hour.
def test_passes_correct_list_to_template(self): other_list = List.objects.create() correct_list = List.objects.create() response = self.client.get(f'/lists/{correct_list.id}/') self.assertEqual(response.context['list'], correct_list)

The fix was to use trailing slash in url. i.e. f'/lists/{correct_list.id}' should always be ending with '/'.
