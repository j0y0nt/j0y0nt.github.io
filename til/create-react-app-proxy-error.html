Error occurred while trying to proxy: localhost:3000/login

I have used create-react-app to create a simple react application & trying to make call to backend api using proxy setting as mentioned in create-react-app documentation. (https://create-react-app.dev/docs/proxying-api-requests-in-development/#configuring-the-proxy-manually) The documenation uses 'http://localhost:5000' for example. However if you use localhost, in your local environment, it will give below error. Error occurred while trying to proxy: localhost:3000/login The solution is to use 127.0.0.1 instead of localhost and you will be able to call the backend without any issue. Example : 'http://localhost:8080/api' must be replaced with 'http://127.0.0.1:8080/api'. AND don't forget to restart the server.
