Quarkus specific commands

Create the Quarkus Project

mvn io.quarkus:quarkus-maven-plugin:create -DprojectGroupId=quarkus -DprojectArtifactId=[NAME]-service -DclassName="{PACAKGE_NAME}" -Dpath="/[PATH]" -Dextensions="resteasy-jsonb, quarkus-hibernate-validator, quarkus-kubernetes, docker, minikube, kubernetes-config"

Add reactive messaging extension for Kafka

quarkus ext add io.quarkus:quarkus-smallrye-reactive-messaging-kafka

Quarkus cli command to create reactive service project. (It is better than maven command)

quarkus create app orgj0:overdraft-service --extension='rest, quarkus-kubernetes, docker, minikube, kubernetes-config, quarkus-messaging-kafka'

Quarkus: Running Reactive Service with Kafka inside minikube cluster

> minikube start --memory=4096
> kubectl create namespace kafka
> kubectl create -f .\strimzi-cluster-operator-0.25.0.yaml -n kafka
> kubectl apply -f .\kafka_cluster.yml -n kafka

Quarkus: Starting Kafka Producer inside minikube cluster

> kubectl -n kafka run kafka-producer -ti --image=quay.io/strimzi/kafka:0.43.0-kafka-3.8.0 --rm=true --restart=Never -- bin/kafka-console-p roducer.sh --bootstrap-server my-cluster-kafka-bootstrap:9092 --topic my-topic

Quarkus: Starting Kafka Consumer inside minikube cluster

> kubectl -n kafka run kafka-consumer -ti --image=quay.io/strimzi/kafka:0.43.0-kafka-3.8.0 --rm=true --restart=Never -- bin/kafka-console-consumer.sh --bootstrap-server my-cluster-kafka-bootstrap:9092 --topic my-topic --from-beginning
