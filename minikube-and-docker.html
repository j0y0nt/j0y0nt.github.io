Frequent Command when working with Quarkus, Kuberenets, Docker & Minikube.

Create the container image for JVM execution

> mvn clean package -Dquarkus.container-image.build=true

On Windows 11 (user quotes in proeprty names):

> mvn clean package -D"quarkus.container-image.build"=true

Starting minikube

> minikube start

To point your shell to minikube's docker-daemon

> minikube -p minikube docker-env --shell powershell | Invoke-Expression

Context issue:

Unable to resolve the current Docker CLI context "default": context "default": context not found

> docker context use default

Deploy to Minikube!

> mvn clean package -Dquarkus.kubernetes.deploy=true

On Windows 11 (user quotes in proeprty names):

> mvn clean package -D"quarkus.kubernetes.deploy"=true

To access service locally on minikube

> minikube.exe service {REPLACE_WITH_SERVICE_NAME}
